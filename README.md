## 📄 Abstract

Online reviews are a critical driver of consumer decisions, but their integrity is compromised by the growing problem of fake reviews. This project presents a comprehensive system for automatically detecting this deceptive content by developing and comparing two distinct methodological approaches. We first establish a performance baseline using classic machine learning models (Random Forest, Logistic Regression) and then implement a state-of-the-art solution using a fine-tuned **BERT** deep learning model. The final system provides a practical, end-to-end pipeline capable of classifying new, unseen reviews with extremely high accuracy.

---

## ✨ Key Features

- **Dual-Approach Modeling:** Implements and compares both traditional Machine Learning and modern Deep Learning techniques.
- **State-of-the-Art Accuracy:** Achieves **98% accuracy** using a fine-tuned BERT model, capable of understanding the deep context of review text.
- **End-to-End Pipeline:** Includes data preprocessing, model training, evaluation, and a ready-to-use prediction function.
- **Model Persistence:** All trained models are saved, allowing for instant inference without retraining.
- **Comprehensive Evaluation:** Detailed performance analysis using Accuracy, Precision, Recall, F1-Score, and Confusion Matrices.

## 🔧 Methodology & Architecture

The system follows a structured pipeline to process, analyze, and classify review text.

 <!-- Replace with your actual system architecture image URL -->

1.  **Data Preparation:** The raw dataset is cleaned, labels are encoded (`OR`/`CG` -> `0`/`1`), and the data is split into training, validation, and test sets.
2.  **Feature Extraction:**
    - **TF-IDF:** For classic models, text is converted into numerical vectors based on word importance.
    - **BERT Tokenization:** For the deep learning model, text is converted into contextual embeddings.
3.  **Model Training:** Classic classifiers (Logistic Regression, Random Forest, etc.) and the BERT model are trained on the prepared data.
4.  **Inference:** The saved models are loaded to classify new, unseen review text and output a prediction (Fake/Real).

## 📊 Dataset

This project uses the **"Fake Reviews Dataset"**, which contains genuine and computer-generated reviews.
- **Labels:**
  - `OR` (Original Review): Genuine, authentic reviews written by real users.
  - `CG` (Computer Generated): Fake, deceptive reviews generated by a computer model.
- **Source:** The dataset is based on the one used in the foundational research paper for this project.

📈 Results & Performance

The fine-tuned BERT model demonstrated significantly superior performance compared to the classic machine learning models.
Model	Accuracy	Key Insight
BERT (Fine-Tuned)	98%	State-of-the-art performance; understands context.
Logistic Regression	86%	Best performing classic model; strong baseline.
Random Forest	84%	Good performance but less effective than linear models.
Naive Bayes	81%	Fast but struggles with nuanced text.
Decision Tree	79%	Prone to overfitting; serves as a basic baseline.
